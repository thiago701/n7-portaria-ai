## Para Experimentar em Casa (5-10 min cada)

**Exercício 1 — Mudar as cores:**
Troque `"blue"` por `"green"` ou `"dark-blue"` no tema e veja a diferença!

**Exercício 2 — Mudar o tema:**
Troque `"System"` por `"Dark"` ou `"Light"` para ver os visuais.

**Exercício 3 — Adicionar um campo:**
Tente adicionar "Observações" no formulário.

## Referência Rápida — Widgets do CustomTkinter

| Widget | Para que serve | Exemplo |
|--------|---------------|---------|
| `CTkLabel` | Mostrar texto | Etiqueta de campo |
| `CTkEntry` | Campo de digitação | Campo para nome |
| `CTkButton` | Botão clicável | Botão "Cadastrar" |
| `CTkFrame` | Agrupar widgets | Caixa do formulário |
| `CTkScrollableFrame` | Lista com rolagem | Lista de moradores |
| `CTkTextbox` | Texto grande | Observações |
| `CTkCheckBox` | Caixa de marcação | "Ativo: sim/não" |

## Curiosidade da Semana

As primeiras interfaces gráficas foram criadas nos anos 1970, no laboratório Xerox PARC. Antes disso, os computadores só tinham tela preta com texto! A Apple se inspirou nesse laboratório para criar o Macintosh em 1984 — o primeiro computador popular com janelas e mouse. Hoje, você usa a mesma ideia para criar a interface do nosso sistema de portaria!
