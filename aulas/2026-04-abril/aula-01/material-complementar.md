## Resumo Rápido - Comandos que Você Vai Usar

Não precisa decorar, mas bom ter a lista à mão!

### Ambiente Virtual

```bash
# Criar ambiente virtual
python -m venv venv

# Ativar (Windows)
venv\Scripts\activate

# Ativar (Mac/Linux)
source venv/bin/activate

# Desativar
deactivate

# Instalar bibliotecas de requirements.txt
pip install -r requirements.txt
```

### Git - Básico

```bash
# Inicializar repositório
git init

# Ver o status dos arquivos
git status

# Adicionar arquivos para o próximo commit
git add .

# Criar um commit com mensagem
git commit -m "Sua mensagem aqui"

# Ver histórico de commits
git log

# Enviar para GitHub
git push origin master
```

## Dicas Importantes

### 1. Guarde os Links
Você vai voltar a estes recursos muitas vezes. Considere criar um arquivo de "Favoritos" com estes links.

### 2. Assista em Português Primeiro
Se não está acostumado com inglês, comece pelos vídeos e artigos em português. Depois explore o conteúdo em inglês.

### 3. Pratique Junto
Não apenas assista/leia. Quando o vídeo mostrar um comando, execute no seu terminal também!

### 4. Um Recurso por Vez
Não precisa consumir tudo hoje. Escolha um vídeo e um artigo. Após a aula, quando tiver tempo, explore os outros.

### 5. Tire Notas
Enquanto assiste, anote os pontos-chave em uma caderneta. Isso ajuda a memorizar!

## Perguntas Comuns

**P: Preciso assistir TODOS esses vídeos?**
Não! Escolha 2-3 que mais te interessam. Se quiser, volte aos outros mais tarde.

**P: E se não entender algo no vídeo?**
Pause, volte, veja novamente. Depois, pergunte para mim na próxima aula. Não deixe dúvida acumulando!

**P: Qual é melhor: vídeo ou artigo?**
Depende de você! Algumas pessoas aprendem melhor com vídeos, outras com texto. Experimente ambos!

**P: Preciso decorar os comandos Git?**
Não! Você vai usar tão frequentemente que vai decorar naturalmente. No início, consulte este material.

**P: E se quiser aprender mais?**
Ótimo! Depois da Aula 01, pode explorar: funções Python, estruturas de dados, programação orientada a objetos. Mas não se apresse!
